## 🚀 GitHub Pages 部署指南

### 步骤 1：创建 GitHub 仓库
1. 登录 GitHub
2. 点击右上角 + → "New repository"
3. 仓库名称：`universal-editor`（或你喜欢的名称）
4. 设为 **Public**（GitHub Pages 免费版需要）
5. 点击 "Create repository"

### 步骤 2：上传文件
**重要：上传此文件夹内的所有文件到仓库根目录**

必需文件：
- `index.html` ← 主页面
- `assets/` 文件夹 ← 包含所有 JavaScript 和 CSS 文件
- `favicon.ico` ← 网站图标
- `README.md` ← 说明文件（可选）

**上传方法：**
- 方法A：拖拽上传（推荐）
- 方法B：使用 Git 命令

### 步骤 3：设置 GitHub Pages
1. 在仓库页面点击 "Settings" 标签
2. 左侧菜单找到 "Pages"
3. 在 "Source" 部分设置：
   - **Source**: Deploy from a branch
   - **Branch**: main（或 master）
   - **Folder**: / (root) ← 重要！选择根目录
4. 点击 "Save"

### 步骤 4：等待部署完成
- 通常需要 5-10 分钟
- 部署完成后会显示绿色勾号
- 网址格式：`https://你的用户名.github.io/仓库名称/`
