## 🚀 GitHub Pages 部署指南

### 步驟 1：創建 GitHub 倉庫
1. 登入 GitHub
2. 點擊右上角 "+" → "New repository"
3. 倉庫名稱：`universal-editor`（或您喜歡的名稱）
4. 設為 **Public**（GitHub Pages 免費版需要）
5. 點擊 "Create repository"

### 步驟 2：上傳文件
**重要：上傳此資料夾內的所有文件到倉庫根目錄**

必需文件：
- `index.html` ← 主頁面
- `assets/` 資料夾 ← 包含所有 JavaScript 和 CSS 文件
- `favicon.ico` ← 網站圖標
- `README.md` ← 說明文件（可選）

**上傳方法：**
- 方法A：拖拽上傳（推薦）
- 方法B：使用 Git 命令

### 步驟 3：設定 GitHub Pages
1. 在倉庫頁面點擊 "Settings" 標籤
2. 左側選單找到 "Pages"
3. 在 "Source" 部分設定：
   - **Source**: Deploy from a branch
   - **Branch**: main（或 master）
   - **Folder**: / (root) ← 重要！選擇根目錄
4. 點擊 "Save"

### 步驟 4：等待部署完成
- 通常需要 5-10 分鐘
- 部署完成後會顯示綠色勾號
- 網址格式：`https://您的用戶名.github.io/倉庫名稱/`
